CF707(2) C 两对和相同的数

发表于 2021-03-16

1500A - Going Home

在一个数组中寻找是否存在互异的四个数abcd，使得a+b == c+d

思路

首先想到的算法就是枚举数组中的每一个数进行判断，这样的时间复杂度是O(n2)看起来会超时，但是实际上根据鸽巢原理，根本不会超时。

因为a[i]<=2.5e6，则a[i]+a[j]<=5e6，根据鸽巢原理，如果存在5e6对不同的i,j，那么其中必然会有一对重复的，因此实际复杂度应该是O(min(n2, 5e6))

假设数组内有n个元素，那么一共会存在 Cn2 * C(n-2)2
