CF716(2)C 模n余1

发表于 2021-05-06

Product 1 Modulo N

[数论] 给定n，在[1..n-1]间寻找一些数，使得这些数的乘积模n余1，输出满足条件的最多数字的情况

思路

假设最后结果的乘积为 P 则有
$$
P\ mod \ n = 1 \
P = K \times n + 1 \ \ (K为整数) \
gcd(P, n) = 1 (根据辗转相除法) \
$$
也就是说 P 和 n 一定是互质的，因此排除掉[1..n-1]区间内所有和 n 非互质的数，将剩下的所有数的乘积记作 T

此时有 $T \ mod \ n = m$

当$m=1$时，显然，取剩下的所有数字即为最终答案。

当$m \not= 1$时，因为$m<n\ 且\ gcd(m, n)=1$，则 m 一定在剩下的数字中，并且根据以下定理
$$
若\ ac \equiv bc(mod m),且(c, m) = d \
则\ a \equiv b(mod \frac m d)
$$
可得：$\frac T m\ mod\ n = 1$,此时，答案为取剩下所有数字，然后把 m 剔除。

晕了，数论实在是接触得太少了，得多碰碰…

代码是真的很简单，主要是思想…
