发表于 2021-05-19 更新于 2021-06-14

Robot Collisions

思路

  1. 奇偶位置上的机器人永远不会相撞，因此将奇偶位置分开处理
  2. 将机器人按其初始位置排序
  3. 从左到右遍历每只机器人
    1. 如果它向右，则将其放入栈中
    2. 如果它向左
      1. 如果栈为空，将它放入栈中（看作反弹后向右的机器人）
      2. 如果栈不为空，则它一定会与栈顶元素相撞（记住我们将奇偶分开处理了）
  4. 如果最后栈中还有元素 那么栈顶的两个机器人会相撞
