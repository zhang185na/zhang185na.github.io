DRF-6: 序列化

1.方法1

这是最原始的序列化工具：

  • 调用values方法，返回字典而不是模型实例对象。每个字典表示一个对象，键对应于模型对象的属性名称。
  • json.dumps()用于将字典形式的数据转化为字符串，json.loads()用于将字符串形式的数据转化为字典
  • ensure_ascii=False 返回值可以包含非ascii字符
2.方法2

自定义序列化类中 source字段用于在数据库中查找对应字段，如果值为一个可调用（callable）的方法，则会调用该方法并返回其返回值，否则直接去数据库中查找对应字段。

DRF-8: 视图

基础的：

  • View

  • APIView(View)

  • GenericAPIView(APIView)

    • from rest_framework.generics import GenericAPIView
class View1View(GenericAPIView):
  queryset = Profession.objects.all()
  serializer_class = ProfSerializer
  pagination_class = PageNumberPagination

  def get(self, request, *args, **kwargs):
      pro = self.get_queryset()
      pg_pro = self.paginate_queryset(pro)
      ser = self.get_serializer(instance=pg_pro, many=True)
      return Response(ser.data)
