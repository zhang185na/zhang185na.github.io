组件的好处

  1. 低耦合高内聚（尽可能类的每个成员方法只完成一件事、减少类内部，一个成员方法调用另一个成员方法）
  2. 单向流数据
  3. 可复用性
  4. 提高更新效率，在更新时，只需要重新渲染有变化的组件

组件通信的方法

  1. 父组件到子组件：通过props的单向数据流
  2. 子组件到父组件：通过$emit自定义事件
